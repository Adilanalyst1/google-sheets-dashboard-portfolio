# 📊 Sales Dashboard

An interactive Sales Dashboard built using **Google Looker Studio** and **Google Sheets** to analyze sales performance, monitor KPIs, and generate business insights from large datasets.

---

## 📌 Overview

This project transforms raw sales data into an interactive dashboard using data analysis and visualization techniques. It enables users to monitor sales performance, profitability, customer trends, and regional performance through dynamic reports.

---

## ✨ Features

* Interactive KPI Scorecards
* Dynamic Filters
* Sales & Profit Analysis
* Top Products & Customers
* Monthly Sales Trends
* Regional Performance Analysis
* Interactive Charts
* Large Dataset Support (100K+ Rows)
